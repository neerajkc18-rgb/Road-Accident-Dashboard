# Road-Accident-Dashboard

The Road Accident Analysis Dashboard is an interactive Business Intelligence solution developed to analyze road accident data and provide meaningful insights into casualties, accident trends, road conditions, and vehicle involvement. The dashboard helps stakeholders identify high-risk areas, monitor accident patterns, and support data-driven safety decisions.

This dashboard was built using Power BI and focuses on visual storytelling through KPIs, trend analysis, geographic mapping, and category-wise comparisons.

Business Insights
Majority of casualties are associated with cars.
Urban areas contribute to a higher percentage of accidents compared to rural regions.
Most accidents occur on single carriageway roads.
Daylight conditions still account for a significant number of casualties.
Monthly trend analysis helps identify seasonal accident patterns.
